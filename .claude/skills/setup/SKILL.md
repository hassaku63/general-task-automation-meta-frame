# setup: 環境セットアップスキル

リポジトリの初回セットアップを行う。
`/setup` を実行するだけで、必要なスキルをユーザー環境にコピーする。

## 使い方
```
/setup
```

## 実行手順

### 1. new-task スキルのインストール

1. リポジトリのルートディレクトリを特定する（CLAUDE.md が存在するディレクトリ）
2. `~/.claude/skills/new-task` が既に存在するか確認する
   - 存在しない → そのままコピー
   - 存在する → リポジトリ側の内容と差分を確認し、差分があればユーザーに上書きするか確認する。差分がなければ「既に最新です」と報告してスキップ
3. `~/.claude/skills/` ディレクトリが存在しない場合は作成する
4. ファイルをコピーする:
   ```bash
   cp -r {リポジトリルート}/skills/new-task ~/.claude/skills/new-task
   ```

### 2. 結果報告

セットアップ結果を報告する:

- コピーしたファイルの一覧
- `/new-task` が使えるようになったこと
- 検証方法: `/new-task test-task` を実行してテンプレートが正しく読み込まれることを確認

### 注意事項

- `~/.claude/skills/new-task/` に既存のファイルがある場合は、差分を確認してから操作する
- スキルを更新したい場合は `/setup` を再実行する（リポジトリ側が正本）
